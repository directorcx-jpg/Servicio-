# CRM ARMOTOR - Gestión Digital CETA

## Descripción
CRM web para el call center CETA de ARMOTOR, grupo automotriz multimarca en el Eje Cafetero de Colombia (Armenia, Pereira, Manizales, Cartago, La Dorada). Marcas: KIA, Honda, FAW, Usados Certificados.

## Arquitectura
- Frontend: HTML/CSS/JavaScript (aplicación web responsive)
- Backend: Google Apps Script (automatizaciones)
- Base de datos: Google Sheets via API
- Hosting: GitHub Pages
- Colores corporativos: Rojo #E53935, Gris oscuro #2C2C3E, Blanco

## Módulos
1. Gestión de Leads Comerciales (2 asesores digitales)
2. Gestión de Casos Taller (5 asesores call center)
3. Cotizador de Mantenimientos
4. Protocolos de Atención - Base de conocimiento integral para asesores que incluye:
   - Guiones de llamada y WhatsApp (por tipo de contacto y marca)
   - Protocolos de atención paso a paso (primera llamada, seguimiento, cierre, escalamiento)
   - Plantillas de mensajes con variables dinámicas ([CLIENTE], [FECHA], [SEDE])
   - Procesos operativos (radicación de casos, agendamiento de citas, manejo de PQR, garantías)
   - Información de referencia rápida (horarios, sedes, contactos internos, extensiones)
   - Checklist de calidad por tipo de gestión
   - FAQ y respuestas a objeciones frecuentes
   - Buscador rápido y sección de favoritos
   - Edición exclusiva del coordinador
5. Dashboard y Reportes

## Reglas de Negocio
- Propiedad de leads/clientes: 20 días por asesor
- Asignación equitativa aleatoria entre asesores del mismo tipo
- Meta primer contacto: menos de 2 horas
- Todo contacto documentado con fecha, hora y observación
- Leads KIA requieren marca de subida a SGC

## Idioma
Todo en español (Colombia)
